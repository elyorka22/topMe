# TopMe Admin Panel

Отдельная админ-панель для управления TopMe.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Особенности

- Управление локациями
- Управление пользователями
- Статистика сайта
- Аналитика

## Примечание

Админ-панель также доступна в основном фронтенд приложении.
Это отдельная версия для независимого деплоя.
